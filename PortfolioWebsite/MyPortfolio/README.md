# Film Portfolio Starter (React)

This project is now set up with:
- React + Vite
- React Router
- Motion (animation)
- Lucide React (icons)
- Tailwind CSS (via Vite plugin)

## Run Locally

Node was installed locally in the repository, so run:

```bash
export PATH="/Users/rux/Projects/portfoliowebsite/.tools/node/bin:$PATH"
cd /Users/rux/Projects/portfoliowebsite/PortfolioWebsite/MyPortfolio
npm run dev
```

## Build

```bash
export PATH="/Users/rux/Projects/portfoliowebsite/.tools/node/bin:$PATH"
cd /Users/rux/Projects/portfoliowebsite/PortfolioWebsite/MyPortfolio
npm run build
```
